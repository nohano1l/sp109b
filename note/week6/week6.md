## gcc編譯流程

![](https://nohano1l.github.io/sp109b/note/week6/picture/1.jpg)

```
直接編譯連結，產生執行檔
$ gcc main.c sum.c -o run    //-o:指定輸出檔
$ gcc main.c sum.c           //輸出檔檔名為a.exe(Windows)、a.out(Mac)

產生組合語言(-S)
$ gcc -S main.c -o main.s

組譯後連結,產生執行檔
$ gcc main.c sum.s -o run

產生目的檔(-c)
$ gcc -c sum.c -o sum.o

連結目的檔,產生執行檔
$ gcc main.o sum.o -o run
```

連結時，會在連結器使用ar、ld

## 堆疊段

* power2.c
```
#include <stdio.h>

int power2(int n){
    if(n==0){
        return 1;
    }
    return power2(n-1)*2;
}

int main(){
    int p=power2(3);
    printf("p=%d\n",p);
}
```
